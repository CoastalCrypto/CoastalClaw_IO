Three Approaches to Coastal Claw

Each solves the architecture differently. Click to select your preference — or mix them.

A

Desktop-First — Electron as Hub

Electron app is the primary product. Web and TUI connect to the running desktop instance.
How it works
┌── Electron App ──────────────┐ │ JARVIS HUD + Core Engine │ │ Voice / Video / GPU │ │ Exposes local port :7474 │ └──────────────┬───────────────┘ │ ┌──────┴──────┐ Web App TUI CLI (companion) (admin) Cloud: Electron on a VM Users access via browser
Setup time for non-technical user
~25-40 min — requires Electron install

✓ Pros

  • Native GPU / mic / camera
  • Rich offline capability
  • Full JARVIS aesthetic control

✗ Cons

  • Cloud story is awkward (VM per user)
  • Heavy download for casual users
  • Updates require re-download
B

Local API Server as Hub — Service-First ⭐

Core runs as a silent local service (like Ollama). All interfaces are thin clients. Cloud = same service on Coastal Crypto servers.
Recommended
How it works
┌── Coastal Claw Service ──────┐ │ Core + Memory + Agents │ │ REST + WebSocket :4747 │ │ Auto-starts on boot │ └──┬──────────┬────────────┬───┘ │ │ │ Web App Electron TUI CLI (browser) (JARVIS) (admin) Cloud: service on CC servers Same web UI, zero install
Setup time for non-technical user
~10-20 min self-hosted · 3 min cloud

✓ Pros

  • All 3 interfaces share live state
  • Cloud = same code, different host
  • Abacus.ai-style web onboarding
  • Silent background updates
  • Ollama/Cursor-style familiarity

✗ Cons

  • Service must be running (auto-start solves)
  • Voice/video needs Electron or browser perms
C

Progressive Web App + Local Bridge

PWA for everyone. Optional local "bridge" binary for local models + GPU. Closest to Abacus.ai's pattern.
How it works
┌── Coastal Crypto Cloud ──────┐ │ Hosted PWA (coastalcrypto) │ │ Accounts + billing + agents │ └──────────────┬───────────────┘ │ optional ┌──────────────┴───────────────┐ │ Local Bridge (tiny binary) │ │ Local models + private data │ └──────────────────────────────┘ TUI: separate power-user tool Electron: optional premium app
Setup time for non-technical user
~3-5 min cloud · 20-30 min with bridge

✓ Pros

  • Absolute lowest entry barrier
  • No install for cloud users
  • Ties directly to Coastal Crypto site

✗ Cons

  • Two separate codebases (cloud + bridge)
  • Local JARVIS experience diluted
  • Complex auth/sync between modes
